#  MERN Stack Airbnb Clone

A **full-stack web application** built using the **MERN stack** — **MongoDB**, **Express.js**, **React.js**, and **Node.js**.  
This project demonstrates how to build a modern web app with a **RESTful API**, **frontend–backend integration**, and **database connectivity**.

---

##  Tech Stack

| Layer | Technology |
|-------|-------------|
| **Frontend** | React.js, HTML5, CSS3, JavaScript (ES6+) |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB (with Mongoose ODM) |
| **Tools & Utilities** | npm, Git, dotenv |
